# AI 搜索 Agent Demo

移动端 AI 搜索卡片 UI 演示项目，展示多轮搜索过程和结果。

## ✨ 核心特性

- **确定性反馈**：实时显示搜索词和参考来源数量，量化搜索进度
- **沉浸式动效**：脉冲动画、数字跳动、平滑展开等微动效
- **分层信息展示**：搜索时自动展开详情，答案生成时自动折叠
- **百度 App 风格**：适配搜索结果卡片场景，简洁实用

## 🎯 设计原则

1. 答案 > 过程：用户要的是答案，过程只是为了建立信任
2. 简洁 > 详尽：少即是多，信息密度要适度
3. 信任 = 来源透明：展示参考来源数量，建立可信度

## 🚀 快速开始

### 本地运行

直接打开 `ai-search-demo/index.html` 即可在浏览器中查看演示。

### 部署到 Vercel

1. Fork 或 Clone 本仓库
2. 在 [Vercel](https://vercel.com) 上导入项目
3. 自动部署完成

或使用 Vercel CLI：

```bash
npm i -g vercel
vercel
```

## 📱 技术实现

- 纯 HTML + CSS + JavaScript，无框架依赖
- 移动端优先设计，适配 375px-480px 屏幕
- 平滑动画和过渡效果
- 模拟真实 AI 搜索流程

## 📂 项目结构

```
ai-search-demo/
├── index.html          # 主页面
├── styles.css          # 样式文件
└── app.js             # 交互逻辑
```

## 🎨 交互流程

1. **搜索开始** → 详情面板自动展开，显示实时搜索进度
2. **搜索进行** → 数字跳动动画，标签逐个出现
3. **答案生成** → 详情面板自动折叠，专注显示答案
4. **手动查看** → 用户可随时点击"详情"查看来源
